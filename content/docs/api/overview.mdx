---
title: API overview
description: How the Metrics API is organized.
---

The Metrics API follows REST conventions over JSON. All requests authenticate with a bearer token
(a Metrics API key created from your Settings). The base URL is `https://api.yourdomain.com`.

The interactive reference is at [/api-reference](/api-reference). The raw OpenAPI 3.1 spec is at
[/openapi.yaml](/openapi.yaml).
