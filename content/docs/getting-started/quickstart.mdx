---
title: Quickstart
description: Sign up, build your first tree, publish it.
---

1. **Sign in** with Google or GitHub at `/signin`.
2. **Create a tree** from the workspace dashboard.
3. **Add nodes** using the floating toolbar or keyboard shortcuts (`M` metric, `F` formula, `S` strategy).
4. **Connect nodes** by dragging from a node's top handle to another node's bottom handle — edges flow UP.
5. **Publish** with the Publish button in the editor topbar. Your tree becomes searchable and
   Viewers can start asking questions about it.
