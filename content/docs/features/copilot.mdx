---
title: Builder Copilot
description: AI assistance embedded in the metric tree editor.
---

Copilot helps Builders with:

- Improving node descriptions.
- Suggesting child metrics for a KPI or goal.
- Drafting formulas.
- Explaining what a metric measures.
- Validating a tree's structure.

Plan gates: Copilot is available on **Pro** (50 requests / day) and **Enterprise** (unlimited).
